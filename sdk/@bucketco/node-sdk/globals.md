### BucketClient

The SDK client.

#### Remarks

This is the main class for interacting with Bucket.\
It is used to update user and company contexts, track events, and evaluate feature flags.

#### Example

```ts
const client = new BucketClient({
  secretKey: "your-secret-key",
});
```
